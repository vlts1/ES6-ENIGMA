HTML & CSS, JS Code Editor. This page will have the current file opened.

It will open the directories from the user's pc (like vscode.dev does)

and will give the user an opportunity to modify the code and save it back

to the pc.

It will include a complete code highlighting system and error message popups,

and a couple more features that I will come up with soon.

Since we will have to use node.js, creating user accounts and saving files online

is an option. With the help of a service like Firebase, accounts can be setup quickly.



There is a menu on the left with the user's open files.
